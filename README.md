ForgeBoard is a modern, Canva-style online whiteboard built using **HTML5 Canvas and Vanilla JavaScript**.  
It features a professional UI and smooth drawing experience inspired by tools like Canva, Miro, and Excalidraw.

This project was developed as part of the **Hephaestusâ€™ Forge Workspace (Whiteboard Tool)** assignment.

---

## Features

- Canva-style professional UI
- Dark theme interface with grid canvas
- Freehand drawing (Pen tool)
- Shape tools (Rectangle, Circle, Line)
- Text tool
- Color picker and stroke size control
- Object-based canvas rendering
- Undo / Redo support
- Save & Load board state (LocalStorage)
- Export whiteboard as PNG image
- No external libraries or frameworks

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- HTML5 Canvas API
- Browser LocalStorage
